ROllertest

Fork of Minetest tracking latest Minetest master (currently 5.8.0-dev), with an aim to improve the Android gameplay experience among other things.

The full source code can be found on GitHub.

Features

  • Android: Improved control scheme
  • Android: Crosshair enabled by default
  • Android: Place with single press
  • Minor main menu improvements
  • Desktop: Discord Rich Presence (ENABLE_DISCORD)
  • And more... (check ROllerozxa's commit history for a full list of changes)

Download

Android

ROllertest is available on Google Play.

Free, no ads, and it will stay that way. Enjoy.
